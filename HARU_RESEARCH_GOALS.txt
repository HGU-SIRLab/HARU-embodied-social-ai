================================================================
[ 연구 비전 — 원문 ]
================================================================

단방향의 기계적 제어를 넘어, 물리적 실체(Physical AI)와 인간-로봇
상호작용(HRI)이 하나로 융합되어 어떤 연령대와도 위화감 없이 교감하는
'체화된 소셜 AI(Embodied Social AI)' 기반의 능동형 반려 로봇 아키텍처를
개발하는 것.

이 로봇은 텅 빈 기계가 아니라 물리 법칙과 보편적 사회 규범 등 확고한
세계관을 '사전 학습된 뇌(Pre-trained Foundation)'로 지니고 출발하되,
세부적인 지식이나 개인의 고유한 성향은 마치 어린아이가 세상을
배워가듯 사용자와의 지속적인 의사소통을 통해 습득하며 동반 성장한다.

특히 단편적인 표정 인식이나 정해진 입출력(I/O)에 의존하지 않고,
사용자의 음성 톤과 억양은 물론 사용자에게 의미 있는 사물과 주변 환경의
맥락(Context)을 소셜 세계 모델(SWM)과 적응형 마음 이론(Adaptive ToM)으로
통합 분석하여, 로봇이 먼저 "이러한 상황 때문에 슬프신 건가요?"라고
선제적으로 다가가는 사전대응적(Proactive) HRI를 지향한다.

이 과정에서 로봇의 추론이 틀렸을 경우 사용자의 직접적인 피드백을
수용하는 인간 참여형(Human-in-the-Loop) 교정 단계를 거치며, 이때 로봇
고유의 자아와 세계관을 잃어버리는 파국적 망각(Catastrophic Forgetting)을
완벽히 방지하기 위해 매일의 상호작용 피드백을 에피소드 단위의 경량화된
LoRA(Low-Rank Adaptation) 어댑터로 모듈화하여 축적한다.

결과적으로 뼈대가 되는 로봇 본연의 정체성은 잃지 않으면서도,
수개월에서 수년에 걸친 상호작용적 강화학습(Interactive RLHF)을 통해
사용자 개인의 삶과 감정에 완벽히 동기화되어 진정한 의미의
'1인 1로봇' 생태계를 여는 따뜻하고 주체적인 동반자를 완성한다.


================================================================
[ 5대 핵심 연구 원칙 ]
================================================================

원칙 1. Pre-trained Foundation
  - 물리 법칙·보편적 사회 규범·감정 상식을 사전 학습된 대형 VLM으로 탑재
  - 사용자 고유 지식은 대화를 통해 점진적으로 습득 (어린아이 성장 모델)

원칙 2. SWM + Adaptive ToM
  - 음성 톤·억양 → 감정 단서 추출 (네이티브 오디오)
  - 사용자에게 의미 있는 사물·환경 맥락 통합 분석
  - 사용자의 현재 정신 상태·숨겨진 의도 모델링
  - 과거 에피소드의 미래 파급 효과 예측

원칙 3. Proactive HRI
  - 명령 대기가 아닌, 로봇이 먼저 다가가는 선제적 상호작용
  - "이러한 상황 때문에 슬프신 건가요?" 수준의 공감적 발화

원칙 4. Human-in-the-Loop
  - 로봇 추론 오류 시 사용자 피드백을 즉각 데이터화
  - 키네스테틱 티칭 (손으로 직접 자세 교정) 지원
  - 언어적·비언어적 피드백 모두 수용

원칙 5. Episodic LoRA (파국적 망각 방지)
  - 베이스 모델(정체성) 고정
  - 매일의 상호작용 → 에피소드 LoRA 어댑터로 모듈화
  - 수개월~수년 누적 → '1인 1로봇' 개인화


================================================================
[ 궁극적 성과 지표 ]
================================================================

  - 어떤 연령대와도 위화감 없는 소셜 교감
  - 수개월~수년 상호작용 후 사용자 개인 삶·감정에 동기화된 반응
  - 파국적 망각 없는 연속적 학습 (베이스 모델 정체성 보존 검증)
  - Proactive HRI 비율 측정 (로봇이 먼저 대화 시작한 비율)
  - HITL 교정 횟수 감소 추이 (학습으로 인한 행동 개선 정량화)

[ 갭을 좁히는 우선순위 로드맵 ]
================================================================

[지금 당장 — 로봇 앞에서]
  1. haru-hitl 로 에피소드 20~30 스텝 수집
  2. python scripts/train_lora.py
  3. brain_node 재시작 → "[Brain] LoRA 어댑터 로드 완료" 확인
  결과: 원칙 1, 4, 5 실증 완료

[단기 — 1~2주]
  4. audio VAD → brain 즉시 추론 트리거 연동
     (60초 주기 제거 → Proactive HRI 실시간화)
  5. Piper TTS 노드 추가 (/haru_speech → 실제 발화)
  결과: 원칙 3 (Proactive HRI) 60% → 85%

[중기 — 1~2개월]
  6. 에피소드 50+ 누적 → HITL 교정 감소율 측정
     (학습 개선 효과 정량화 → 논문 핵심 실험)
  7. DPO 적용 (Direct Preference Optimization)
     SFT 데이터 기반 선호도 쌍 구성 → 진정한 RLHF
  결과: 원칙 5 (Episodic LoRA) 실증 + 논문 Figure

[장기 — 석사 논문 마무리]
  8. 종단 평가 (3~6개월 상호작용 후 적응도 측정)
  9. 파국적 망각 방지 정량 검증 (Continual Learning 벤치마크)
 10. '1인 1로봇' 생태계 데모
